# Subway_Congestion
Simple project to bridge multiple devops technologies

* database: PostgreSQL
* API: Go, Rust, RESTful
* simulation: Python
* Containerization: Docker, Kubernetes
* analysis: Apache-Spark
* Pipeline: Apache-Kafka, Apache-Airflow
* CI/CD: Jenkins
* Cloud: Amazone Web Service
* OS: Linux
* IaC: Terraform
* Load Balancing: Nginx
